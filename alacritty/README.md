## Installation

1. Replace the color section of your alacritty.yml with the following code block.

        colors:
        primary:
            background: '#1C1E26'
            foreground: '#b2b2c2'
        cursor:
            text: '#1C1E26'
            cursor: '#b2b2c2'
        normal:
            black: '#2E303E'
            red: '#E95678'
            green: '#29D398'
            yellow: '#FAB795'
            blue: '#26BBD9'
            magenta: '#EE64AE'
            cyan: '#59E3E3'
            white: '#FDF0ED'
        bright:
            black: '#16161C'
            red: '#EC6A88'
            green: '#3FDAA4'
            yellow: '#FBC3A7'
            blue: '#3FC6DE'
            magenta: '#F075B7'
            cyan: '#6BE6E6'
            white: '#FDF0ED'

2. Restart Alacritty.
3. You're done.
